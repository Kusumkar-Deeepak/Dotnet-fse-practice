# Exercise 05 - SQL Functions

## Objective

To understand and implement different types of User Defined Functions (UDFs) in SQL Server.

---

## Concepts Covered

1. Scalar Functions
2. Table Valued Functions
3. Function Modification
4. Nested Functions
5. Function Deletion

---

## Files

### setup.sql

Creates and verifies required tables and sample data.

### 01_scalar_function.sql

Creates a scalar function to calculate annual salary.

### 02_table_valued_function.sql

Creates a table-valued function to retrieve employees by department.

### 03_bonus_function.sql

Creates a function to calculate employee bonus.

### 04_modify_bonus_function.sql

Modifies the bonus calculation logic.

### 05_delete_bonus_function.sql

Deletes the bonus function.

### 06_execute_annual_salary.sql

Executes annual salary function.

### 07_specific_employee_salary.sql

Calculates annual salary for a specific employee.

### 08_finance_department.sql

Retrieves employees from a specific department.

### 09_nested_function.sql

Creates a function that uses multiple functions together.

### 10_modify_nested_function.sql

Executes and verifies nested function results.

---

## Learning Outcomes

- Understand Scalar Functions.
- Understand Table-Valued Functions.
- Learn function modification.
- Learn nested function implementation.
- Learn function execution and deletion.

---

## SQL Concepts Used

- CREATE FUNCTION
- ALTER FUNCTION
- DROP FUNCTION
- RETURN
- TABLE VALUED FUNCTION
- SCALAR FUNCTION

---

## Result

Successfully implemented and tested various SQL Server functions.
