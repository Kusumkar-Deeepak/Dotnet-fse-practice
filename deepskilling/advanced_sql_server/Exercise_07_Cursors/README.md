# Exercise 07 - Cursors

## Objective

To understand row-by-row data processing using SQL Server cursors.

---

## Concepts Covered

1. Basic Cursor
2. Static Cursor
3. Dynamic Cursor
4. Forward Only Cursor
5. Keyset Cursor

---

## Learning Outcomes

- Understand row-by-row processing.
- Learn cursor lifecycle.
- Compare different cursor types.
- Understand cursor advantages and limitations.

---

## SQL Concepts Used

- DECLARE CURSOR
- OPEN
- FETCH
- WHILE
- CLOSE
- DEALLOCATE

---

## Cursor Types

### Static Cursor

Creates a snapshot of data.

### Dynamic Cursor

Reflects real-time changes.

### Forward Only Cursor

Allows movement only in forward direction.

### Keyset Cursor

Keeps row identity fixed while allowing updates.
