## Stored Procedures Created

### sp_GetEmployeesByDepartment
Returns employees from a department.

### sp_CountEmployees
Returns employee count.

### sp_TotalSalary
Returns total department salary using OUTPUT parameter.

### sp_UpdateEmployeeSalary
Updates employee salary.

### sp_GiveBonus
Adds bonus to employee salary.

### sp_UpdateSalaryTransaction
Updates salary using transaction.

### sp_SearchEmployee
Uses dynamic SQL for flexible searching.

### sp_UpdateSalarySafe
Handles errors using TRY...CATCH.

## Learning

Stored Procedure:
A reusable SQL program stored inside the database.

Benefits:
- Better Performance
- Reusability
- Security
- Easier Maintenance
