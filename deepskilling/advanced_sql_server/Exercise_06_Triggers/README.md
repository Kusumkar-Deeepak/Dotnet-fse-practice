# Exercise 06 - Triggers

## Objective

To understand how SQL Server triggers automate actions when database events occur.

---

## Concepts Covered

1. AFTER Trigger
2. INSTEAD OF Trigger
3. LOGON Trigger
4. Trigger Modification
5. Trigger Deletion
6. Automatic Data Updates using Triggers

---

## Files

### setup.sql

Verifies existing Employees and Departments tables.

### 01_after_trigger.sql

Logs employee salary changes automatically.

### 02_instead_of_delete_trigger.sql

Prevents accidental employee deletion.

### 03_logon_trigger.sql

Demonstrates server-level login trigger.

### 04_modify_trigger.sql

Modifies an existing trigger.

### 05_delete_trigger.sql

Removes a trigger from the database.

### 06_annual_salary_trigger.sql

Automatically updates annual salary when salary changes.

---

## Learning Outcomes

- Understand trigger execution.
- Learn auditing using triggers.
- Prevent unauthorized actions.
- Automatically maintain data consistency.
- Learn trigger modification and deletion.

---

## SQL Concepts Used

- CREATE TRIGGER
- ALTER TRIGGER
- DROP TRIGGER
- AFTER UPDATE
- INSTEAD OF DELETE
- INSERTED Table
- DELETED Table

---

## Result

Successfully implemented and tested different types of SQL Server triggers.
