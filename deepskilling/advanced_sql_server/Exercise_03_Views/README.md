# Exercise 03 - Views

## Concepts Covered

1. Simple View
2. Full Name View
3. Annual Salary View
4. Employee Report View

## Views Created

### vw_EmployeeBasicInfo

Displays employee details with department name.

### vw_EmployeeFullName

Combines FirstName and LastName.

### vw_EmployeeAnnualSalary

Calculates annual salary using:

AnnualSalary = Salary \* 12

### vw_EmployeeReport

Displays:

- EmployeeID
- Full Name
- Department Name
- Annual Salary
- Bonus

## Learning

View is a virtual table created using a SELECT query.

Benefits:

- Reusability
- Security
- Simpler Queries
