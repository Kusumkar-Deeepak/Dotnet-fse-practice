# ASP.NET Core 8.0 Web API Tasks

This folder contains three separate small projects for the Web API tasks.

## Folder map

- `FirstWebAPI` - task 1, basic Values controller, read and write actions.
- `Task2SwaggerWebAPI` - task 2, Swagger demo, Postman demo, route name example.
- `Task3CustomModelWebAPI` - task 3, custom Employee model, custom filters, exception logging.

## Notes

- Each project is kept small and simple.
- Swagger is used to show the endpoints neatly in the browser.
- The code is written in a student-friendly style with only the required pieces.

## How to run the three tasks

1. Open the project folder in VS Code or Visual Studio.
2. Open the required task folder.
3. Restore packages if needed.
4. Run the project from that folder.
5. Open the browser or Swagger page for the project.
