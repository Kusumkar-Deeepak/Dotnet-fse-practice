# Exercise 1 - Ranking and Window Functions

Concepts Used:
- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- OVER()
- PARTITION BY

Learning:

ROW_NUMBER:
Gives unique rank.

RANK:
Same rank for ties and skips next rank.

DENSE_RANK:
Same rank for ties without skipping.

Business Problem:
Find top 3 expensive products in each category.
